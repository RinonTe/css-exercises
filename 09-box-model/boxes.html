An essay on box-model

Box model

It is said that all web developers should know what box model is and it is really important to understand how it works. We are going to learn about it in this essay.

Firstly, let's define what box model is. It is mostly about designing and layout in CCS . The CCS. Box model is the box that wraps around every html element.

We need to understand it otherwise we will not be able to create layouts with CCS or to align items with other items. The browser creats a box for each element according to the atandard CCS basic model but we control the size, position, and properties of the box by using CCS.

Every box is contained of four areas, defined by their respective edges: the content edge, padding edge, border edge, and margin edge.
